# vip-macaujck-guide

本仓库用于归档和发布多个独立 HTML 页面，作为通用资料存放与索引用途。  
仓库名称中的 `vip-macaujck-guide` 仅为标识符，不代表指向任何特定域名或网站。

## 项目简介

这是一个以 HTML 页面归档为主的仓库，收录了若干独立制作的静态页面，内容涵盖信息整理、工具展示、引导说明等通用主题。  
所有页面均为独立构建，不依赖后端服务或数据库，可直接通过浏览器打开使用。  
本仓库不针对任何特定域名、商业品牌或服务，仅作为技术存档与页面发布用途。

## 目录说明

```
├── index.html          # 主入口页面（可选）
├── pages/              # 存放独立 HTML 页面的文件夹
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录为页面归档核心，每个 HTML 文件为一个独立单元，可直接访问。
- `assets/` 目录存放页面所需的公共资源，按类型分文件夹管理。
- 主入口 `index.html` 可选，用于展示页面列表或导航。

## 页面归档说明

- 所有 HTML 页面均为静态文件，无后端交互，适合直接部署到 GitHub Pages 或其他静态托管服务。
- 页面内容以通用信息整理、工具说明、引导参考为主，不包含任何具体域名、网址、营销承诺或诱导点击内容。
- 每个页面均保持独立性与可读性，方便单独引用或嵌入。

## 维护说明

- 欢迎通过 Issue 或 Pull Request 提出建议，但请注意本仓库不接受任何涉及具体商业推广或违规内容的提交。
- 如发现页面存在显示问题或链接错误，请提供详细描述以便修复。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 目录下的任一 HTML 文件。
3. 也可以将整个仓库部署到 GitHub Pages 或其他静态服务器，以获得在线访问地址。
